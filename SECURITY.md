# Security Policy

## Supported Versions

Only the latest version of Armadarr is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not open a public issue. Instead, please report it via GitHub's private vulnerability reporting feature or contact the maintainers directly.

We will acknowledge your report within 48 hours and provide a timeline for a fix if the vulnerability is confirmed.
